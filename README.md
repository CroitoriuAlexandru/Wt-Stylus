# Keyboard Shortcuts

### Global keyboard shortcuts
ctrl + a = Create Sibling After
ctrl + s = Create Sibling Before
ctrl + z = Create Child First
ctrl + x = Create Child Last
ctrl + c = Copy Node
ctrl + v = Paste Node
ctrl + d = Delete Node

7 = Toggle Left Panel
9 = Toggle Right Panel

### Commands Panel
ctrl + z = select template
ctrl + x = select Style
